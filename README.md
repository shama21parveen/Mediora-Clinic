# Mediora Clinic

Mediora Clinic is a modern and responsive clinic landing page built with React, Vite, and Tailwind CSS. The website presents clinic services, doctor information, patient journey, testimonials, contact details, and an appointment form.

## Features

- Responsive clinic landing page
- Hero section with call-to-action
- Doctor introduction section
- Healthcare services section
- Patient journey cards
- Patient experience/testimonial cards
- Clinic story video section
- Visit clinic contact and map section
- Get in Touch appointment form
- Footer with social links

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript
